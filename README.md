# BERT Sentiment Analysis Dashboard

Dashboard interaktif untuk visualisasi hasil penelitian analisis sentimen menggunakan model BERT.

## Fitur

- Halaman beranda dengan bagian untuk abstrak dan latar belakang penelitian
- Visualisasi dataset mentah dan berlabel
- Halaman preprocessing untuk memahami langkah-langkah pembersihan data
- Halaman model dan training untuk melihat arsitektur model dan proses pelatihan
- Halaman hasil dan evaluasi dengan metrik-metrik performa model
- Visualisasi interaktif seperti word cloud dan frekuensi kata
- Halaman kesimpulan untuk menulis refleksi dan kesimpulan penelitian
